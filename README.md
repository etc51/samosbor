# samosbor

Рабочая копия проекта находится в `D:\projects\samosbor`.

Репозиторий на GitHub: `https://github.com/etc51/samosbor`

Сейчас это минимальный стартовый каркас:

- репозиторий разворачивается и ведется локально на диске `D:`
- изменения коммитятся локально и отправляются в `origin`
- структура специально оставлена нейтральной, чтобы дальше развивать проект без лишнего шаблонного мусора
